- *classmethod* chain.from_iterable(*iterable*): 문자열 반복
```python
# from itertools import chain
def from_iterable(iterables):
    # chain.from_iterable(['ABC', 'DEF']) --> A B C D E F
    for it in iterables:
        for element in it:
            yield element
```
